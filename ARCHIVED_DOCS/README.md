# Archived Documentation

This folder contains deployment guides for the previous cPanel + MySQL hosting setup, which has been **deprecated** in favor of Supabase + Netlify.

These files are kept for historical reference only and should not be used for new deployments.

- `DEPLOYMENT_CHECKLIST.md` - Original cPanel deployment checklist
- `DEPLOYMENT_QUICK_REFERENCE.md` - Original cPanel quick reference
- `DEPLOYMENT_REVIEW.md` - Original cPanel deployment review
- `QUICK_START_DEPLOYMENT.md` - Original cPanel quick start
- `README_DEPLOYMENT_START_HERE.md` - Original cPanel deployment start guide
- `INSTALLER_GUIDE.md` - Original MySQL installer guide
- `INSTALLER_COMPLETE_SUMMARY.md` - Original installer summary
- `INSTALLER_IMPLEMENTATION_SUMMARY.md` - Original installer implementation summary
- `deploy.sh` - Original cPanel deployment script
